# 📊 Data Cleaning & Visualization Project

## Overview
This project involves cleaning the SuperStore Sales dataset and creating an interactive Power BI dashboard to analyze sales performance and business insights.

## Tools Used
- Microsoft Excel
- Microsoft Power BI

## Files
- `SuperStore Sales DataSet.xlsx` – Sales dataset
- `sales.pbix` – Power BI dashboard
- `sales dashboard.png` – Dashboard preview
